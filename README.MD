 Projeto API RESTful - Gerenciamento de Usuários
Este projeto implementa uma API RESTful completa para gerenciar a entidade Usuários, utilizando Node.js, Express e MongoDB. O objetivo é cumprir todos os requisitos de um projeto Full Stack, incluindo CRUD, autenticação JWT, validações e testes unitários.

 Status do Projeto
 TODOS OS REQUISITOS CUMPRIDOS E TESTES UNITÁRIOS APROVADOS (9/9).

Stack Tecnológico
Backend Framework: Node.js (v20+) e Express

Banco de Dados: MongoDB Atlas (Mongoose ODM)

Segurança: JSON Web Token (JWT) e Hashing de Senhas (bcrypt)

Qualidade: Jest e Supertest
